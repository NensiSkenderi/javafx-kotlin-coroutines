# javafx-kotlin-coroutines

## About The Project

A simple project to show the use of coroutines in Kotlin with JavaFx and making callbacks to JavaFx elements
based on coroutine state.

## Technologies used: 

  * Kotlin
  * JavaFX
  * Kotlinx Coroutines JavaFX
  * Gradle's Kotlin DSL
